15.使用ubuntu和shadowsocks搭建国外vpn实现翻墙

发表于 2017-01-22   |   分类于 others   |     |   阅读次数

　　今天来说说，如何在ubuntu上搭建vpn服务器，让我们可以愉快的翻墙！

前期准备

　　这里我们用ubuntu + shadowsocks来搭建我们的vpn服务器进行翻墙

　　首先，我们需要一个国外的ubuntu服务器，这里我用的是搬瓦工（bandwagon），附上地址：http://banwagong.cn/

　　还是挺便宜的，比阿里便宜很多，当然，土壕请忽略哈！

　　这是我的服务器

　　这里要说下，如果你搬瓦工用的centos的系统（系统可以任意重装的），他的web端管理后台提供一键安装shadowsocks的功能，非常方便，不过我们今天主要讲ubuntu的安装哈。

　　接下来，我们先下个shadowsocks的客户端，地址：http://pan.baidu.com/s/1nuAcrwP 4k6u

安装shadowsocks

　　搬瓦工系统刚装完，要更新获取一下源，ubuntu直接

apt-get update & apt-get upgrade

　　完成之后，我们开始安装shadowsocks服务端，首先安装pip

apt-get install python-pip

　　接着就可以用pip来安装shadowsocks服务端了

pip install shadowsocks

配置启用shadowsocks服务端

　　安装非常快的，秒好，剩下的就是配置了，在etc下创建一个文件，保存配置信息

vim /etc/shadowsocks.json

　　内容如下：

{
    "server": "10.10.10.10",
    "server_port": "8388",
    "local_address": "127.0.0.1",
    "local_port": "1080",
    "port_password": {
        "8387": "password"
    },
    "timeout": 300,
    "method": "aes-256-cfb",
    "fast_open": false
}

　　server是你的服务器外网IP，server_port是你服务器开通该服务的端口号，local_address是本地代理的地址，local_port是本地代理的端口号，port_password是创建客户端账号，其中8387是端口号，password是密码，这里可以创建多个账号，供多人连接，再往下，timeout是超时时间，method是加密方式，从客户端那里可以看到有多种加密方式，fast_open是提供是否使用TCP连接。

　　配置好文件之后，我们就启动它

ssserver ‐c /etc/shadowsocks.json ‐d start

客户端连接

　　服务端shadowsocks启用好了，现在我们尝试用客户端连接试试，注意信息要跟服务器配置一致，加密方式也要一致

　　在任务栏中右键我们的shadowsocks客户端（纸飞机图标），选在启用系统代理

　　图标由灰色变白色即可

　　最后我们测试下，是否真的连上了，百度ip，成功了！

　　访问fackbook

　　注意浏览器要设置本地代理，代理端口就是之前设置的本地代理端口，这里我用chrome插件实现代理
