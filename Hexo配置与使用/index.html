发表于 2016-03-07   |   分类于 others   |     |   阅读次数

　　本文主要介绍Hexo的一些基本站点配置，主题的选择和使用，以及文章的撰写发表！

域名绑定

　　首先我们的博客域名angoddess.github.io是访问正常的，但是我们可能觉得这个域名太丑了，简直弱爆了。我们想如果有人访问angoddess.github.io时指向别的域名，但还是正常访问我们的博客，这就是我们常说的重定向，操作很简单，首先我们要申请个自己的域名。
　　我是在阿里云申请的域名：www.angoddess.cn，地址是：http://wanwang.aliyun.com/domain/
　　申请好域名之后，我们要配置让这个域名能够解析到我们的GitHub博客的地址，也就是访问www.angoddess.cn会访问到我们的博客内容。
　　我们来到阿里云的管理控制台，域名，找到我们的域名，选择解析

　　我们添加一条CNAME记录，把域名解析到我们博客的位置，如图：

　　启用后，我们访问www.angoddess.cn就可以访问到我们的博客了
　　PS：可能启用后，马上访问还是没效果，不要急，等待几分钟，或者直接清理浏览器的缓存再访问试试。
　　如果想访问angoddess.cn也访问到我们的博客，那么就再添加一条CNAME记录，主机记录那里改为@，其它不变就可以了。
　　但是，我们会发现我们访问angoddess.github.io域名时，域名依然没有变。因为我们只是让www.angoddess.cn有内容了而已，还没有让博客域名重定向到这个域名。
　　别急，接下来我们就来重定向博客域名，步骤很简单，一步解决。
　　我们只需要在Hexo下的source目录中，添加一个文件名为CNAME的文件，然后内容就是你的域名即可，如图：

　　最后更新代码到你的GitHub上。
　　这时我们访问原来的博客地址就会重定向到我们在阿里云申请的域名啦，而且访问正常！是不是很酷！
　　PS：如果没有马上生效，记得不要急，等几分钟或清理缓存。

主题使用与配置

Hexo的默认主题是landscape，其实也还蛮不错的啦，如果不喜欢，我们在GitHub上可以找到很多其他的主题，这里介绍一款较受欢迎的主题next，GitHub地址：https://github.com/iissnan/hexo-theme-next
可以在GitHub上面直接下载zip文件，然后解压到我们自己的Hexo根目录下的themes下（themes就是存放我们的主题的文件夹，主题名就是文件夹名），命名为next。
