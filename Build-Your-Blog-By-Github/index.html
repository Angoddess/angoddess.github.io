1.Build Your Blog By Github

发表于 2016-03-01   |   分类于 others   |     |   阅读次数

Here I will teach you how to bulid your personal blog by github.
Tools: git | github | hexo

My System:window 7 64bit

Frist: Register your github user.

Input your email and password,then follow the tips to do.Just so easy!

Second: Build a new repository for your blog.

Login your github and build a new repository on the right top.

Input the information about of your blog.

then,click the button “Create repository” to create your repository.
Now,select the Setting tab and click the button “Launch automatic page generator” to design your blog page.

Here you can design your blog index page.

Select the style you love for your blog page.

Finally,pulish your page to finish your blog page.
Now,you already have a blog by yourself.
You can visit by http://username.github.io(e.g.http://angoddess.github.io).

Third: Install the git and connect to your github.

Install the git is too easy so I will not teach it. Git Download
Now we make the git to connect your github by ssh.
Open your git bash(git shell).
Set your username and email:

Tip:This name and email will use when your update your blog code into github.
You can check your set.

Now begin to create your ssh key.
Check the folder “.ssh” exists.
windows addr: C:\Users\angoddess.ssh
linux addr: ~/.ssh
Delete this folder and create your key.

Follow the tip to enter, you will get two file “id_rsa” and “id_rsa.pub”.
Now open the id_rsa.pub to copy the ssh key and put it on your github.

Write your key title and key value to add.

Finally, you can test it.

Fourth: Install the hexo and get all file to your workspace.

Install hexo must to install node. Node Download
Open git shell.

Create your workspace’s folder and get hexo.

Finally: Test the blog on localhost and update it into github.

Generate static pages.

Test the blog on localhost.


Update it into github must be set deploy.
Open file “_config.yml”.
Addr: D:\git\github\Hexo_config.yml
Rewrite the deploy like this.

Here you will input your username and email.
